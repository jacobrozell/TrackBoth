## Product promise (1.0.0)

TrackBoth helps people **log habits and vices daily**, **see progress over time**, and **stay motivated** — with local data they control via JSON export.

**Subtitle:** Habits & Vices  
**One sentence:** Daily boolean tracking for habits and vices, with goals on rows, history, charts, motivations, and settings. **Free always.**

---

## Locked core loop

These five tabs ship in 1.0.0 Release. **Goals tab is dev-only** — goal progress shows on Track rows.

| Surface | Ships | Core behavior |
|---------|-------|---------------|
| **Track** | ✅ | Daily log; habits/vices sections; week calendar with completion dots; hero streaks; milestone banners; row metadata (goals, savings, slip timer); LoggingSheet |
| **History** | ✅ | Calendar + list; edit entries; filters |
| **Motivation** | ✅ | Feed + primary motivation for vices (I Am Sober-style accountability) |
| **Charts** | ✅ | Line, bar, heatmap (quantity chart dev-only) |
| **Settings** | ✅ | Export/import JSON, delete all data, themes (2), week start, share app |

**Onboarding** ships with habit/vice presets; skip creates placeholder rows (`My habit` / `My vice`).

**Vice-specific core** (Track/History): avoided/not-avoided semantics, slip timer, money-saved estimate, slip motivation in LoggingSheet.

## Iteration mode (now → ship)

### Do iterate on

1. **Correctness** — vice vs habit semantics; streak boundaries; export round-trip
2. **Polish** — device/iPad, Dynamic Type, VoiceOver on Charts + Motivation
3. **Release ops** — TestFlight, dogfood, App Store copy, hosted privacy URL
4. **Bug fixes** — P0/P1 only within locked scope

### Do not iterate on (until after 1.0)

- New tabs or major navigation changes
- Widget embed, watch, notifications, shortcuts
- CloudKit sync
- Gamification
- Schema changes without baseline doc update
